## 🧠 What's Inside

| Language | Topics Covered |
|----------|-----------------|
| **C** | Variables, data types, operators, conditionals, loops, functions, recursion, pointers, arrays (1D & 2D), strings, structures, file handling |
| **C++** | Hello world, datatypes, `const`, namespaces |
| **Python** | Variables, data types, input/output, and chapter-wise practice problems |
| **HTML** | HTML tag basics, page structure, and core elements |

Each language folder is broken down into **chapters** (`CHAPTER_XX` / `Chapter X`), with a matching **practice set** folder (`*_PRACTICE_SET` / `Chapter X - PS`) containing solved problems for that chapter.

## 🚀 Getting Started

### Running C files
```bash
gcc filename.c -o output
./output
```

### Running C++ files
```bash
g++ filename.cpp -o output
./output
```

### Running Python files
```bash
python3 filename.py
```

### Viewing HTML notes
Open the `.md` files in any Markdown viewer, or view them directly on GitHub.
